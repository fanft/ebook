                   【知识拓展】
                   替代词 one 和 ones 的用法

                   1.   one 只能替代单数名词，one 的复数形式 ones 只能替代复数名词。
                   I have lost my pen.  I’m going to buy one. 我的钢笔丢了。我要再买一支。
                    2.   替代词 one 或 ones 和它们所替代的名词中心词所指不一定是同一对象。
                   例：I have a new coatand several old ones.我有一件新外套和几件旧的。
                   3.   当替代词 one 或 ones 带有后置修饰语时，它们前面总有定冠词。

                   例：Our new TV is moreexpensive than the one we had before.
                   我们的新电视机比我们之前的那台要贵得多。
                   4.   当替代词 one 或 ones 在形容词比较级、最高级以及某些代词如 this，that，
                   which 和序数词等之后时，可以省略。

                   例：This book is muchbetter than that (one). 这本书比那本好得多。
                   5.   替代词 one 或 ones 通常不用在物主代词和名词所有格之后。如不可说 my
                   one(s), your one(s), Peters one(s)等，one 或 ones 也不能用在 own 之后但是，如
                   果有了形容词，one(s)可以和物主代词及名词所有格连用。

                   例：My cheap cameraseems to be just as good as John s expensive one. 我的便宜
                   的相机看起来和约翰的贵的那个一样好。
